# 🦦 카피바라 코드 편집기

## 📖 개요

카피바라 코드 편집기는 쉽고 재미있게 사용할 수 있는 커스텀 언어인 "카피바라 코드"를 작성하고 실행할 수 있는 간단하고 인터랙티브한 애플리케이션입니다. 이 편집기를 사용하면 사용자는 카피바라 코드 스크립트를 작성하고, 열고, 저장하고 실행할 수 있습니다.

---

## ⚙️ 기능

* 인터랙티브 편집기: 카피바라 코드를 작성하고 편집할 수 있는 텍스트 편집기
* 파일 처리: `.capy` 파일 저장 / 불러오기
* 코드 실행: 편집기에서 직접 실행 가능
* 커스텀 명령어: 다양한 기능을 수행하는 전용 문법 제공

---

## 📥 설치

👉 Releases에서 EXE 다운로드 (Portable)

* 설치 필요 없음
* 실행 파일만 실행하면 바로 사용 가능

# 🧠 카피바라 코드 문법

카피바라 코드는 반드시 아래 구조를 따라야 합니다:

```plaintext
<카피바라~>
코드
<카피바라!>
```

---

## 📌 명령어 설명

---

### 🔹 출력

* `말해바라~ "text"`
* `보여바라~ "text {변수}"`

```plaintext
<카피바라~>
말해바라~ "안녕하세요!"
보여바라~ "값: {숫자}"
<카피바라!>
```

---

### 🔹 입력 및 변수

* `적어바라~ "text"`
* `변수바라~ "변수명"`

```plaintext
<카피바라~>
적어바라~ "이름을 입력하세요"
변수바라~ "이름"
보여바라~ "{이름}님 환영합니다"
<카피바라!>
```

---

### 🔹 계산

* `계산바라~ "수식 = 변수"`

```plaintext
<카피바라~>
계산바라~ "3 + 5 = 결과"
보여바라~ "{결과}"
<카피바라!>
```

---

### 🔹 조건문

* `만약바라~ (조건)`
* `아니면만약바라~ (조건)`
* `아니면바라~ "내용"`

```plaintext
<카피바라~>
적어바라~ "숫자 입력"
변수바라~ "숫자"

만약바라~ ({숫자} > 10)
말해바라~ "10보다 큼"

아니면만약바라~ ({숫자} == 10)
말해바라~ "10임"

아니면바라~ "10보다 작음"
<카피바라!>
```

---

### 🔹 반복문

* `반복바라~ "횟수"` ~ `끝바라!`
* `루프바라~ "횟수"` ~ `루프바라!`

```plaintext
<카피바라~>
반복바라~ "3"
말해바라~ "반복 중"
끝바라!
<카피바라!>
```

---

### 🔹 랜덤

* `랜덤바라~ "최소, 최대, 변수"`

```plaintext
랜덤바라~ "1, 10, 숫자"
보여바라~ "{숫자}"
```

---

### 🔹 리스트

* `목록바라~ "이름"`
* `넣어바라~ "리스트, 값"`
* `섞어바라~ "리스트"`
* `골라바라~ "리스트, 변수"`
* `길이바라~ "리스트, 변수"`

```plaintext
<카피바라~>
목록바라~ "음식"
넣어바라~ "음식, 피자"
넣어바라~ "음식, 치킨"

골라바라~ "음식, 결과"
보여바라~ "{결과}"
<카피바라!>
```

---

### 🔹 함수 출력

* `함수말해바라~ "텍스트 {함수}"`

```plaintext
<카피바라~>
함수말해바라~ "합계: {sum(1,2)}"
<카피바라!>
```

---

### 🔹 기타 명령어

```plaintext
기다려바라~ "초"
열어바라~ "URL"
명령바라~ "명령어"
저장바라~ "파일, 내용"
시간바라~
날짜바라~
청소바라~
꺼져바라~
```

# 🧪 예제 코드

```plaintext
<카피바라~>
계산바라~ "입력값들 = []"

루프바라~ "5"
    적어바라~ "값을 입력하세요:"
    변수바라~ "입력값"
    함수말해바라~ "입력된 값: {입력값}"
    계산바라~ "입력값들.append('{입력값}')"
루프바라!

말해바라~ "모든 입력된 값:"
보여바라~ "{입력값들}"

말해바라~ "평균 계산"

계산바라~ "sum(입력값들)/len(입력값들) = 평균"

보여바라~ "평균: {평균}"

만약바라~ ({평균} > 50)
말해바라~ "높음"

아니면바라~ "낮음"

랜덤바라~ "1, 100, 랜덤값"
보여바라~ "랜덤값: {랜덤값}"
<카피바라!>
```
